```


    Breakpoint-->        Extra small     Small       Medium      Large   Extra large     Extra extra large

Class infix-->       None            sm          md          lg          xl              xx1     

Dimensions           <576px          2576px      2768px      2992px      21200px         1400px
```

these are the breakponts on which the screensizes have to break for the
 smooth and  best adaptive way for the website to be fit in all the screensizes.

@include media-breakpoint-up(sm) { ... }
@include media-breakpoint-up(md) { ... }
@include media-breakpoint-up(lg) { ... }
@include media-breakpoint-up(xl) { ... }
@include media-breakpoint-up(xxl) { ... }

behind the sceen in SASS file these breakpoints are included as an Media Queries.
